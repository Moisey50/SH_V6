InflateRect gadget
Input Formats:CRectFrame
Outputs Formats:CRectFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Data\Rectangles\ for grouped by type or Data\Rectangles\ by default

InflateRect gadget change the sizes of received rectangle. The received rectangle dimensions are enlarged (or diminished) according to ‘X’ and ‘Y’ values. The chaged rectangle is sent to output pin.

 

Gadget setup dialog has following parameters:

X - sets the horizontal enlargement, X to the left and X to the right. Possible values: -50 - 50.

Y - sets the vertical enlargement, Y up and Y down. Possible values: -50 - 50.
