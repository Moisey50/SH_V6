Video_{Operation} gadgets
 
   Input Formats:      pin1: CVideoFrame, pin2: CVideoFrame

Outputs Formats:  CVideoFrame

Duplex Formats:    NONE

Location in gadget tree: Filters\Video\Logic, Filters\Video\Math for grouped by type or Video\Logic, Video\Math by default

Video_{Operation} gadgets is set of gadgets which makes simple logic or math operation with entire two video frames. Both frames must have same sizes.

All family of filters is tabulated below:

Filter NameOperation performed with each point of the images
Video_XORout(x,y)=abs(in1(x,y)-in2(x,y)
Video_XORBout(x,y)= ((in1(x,y)<maxI/2) XOR (in2(x,y)<maxI/2))?maxI:0, where maxI - maximal value of intensity
Video_ANDout(x,y)=min(in1(x,y),in2(x,y))
Video_ANDBout(x,y)= ((in1(x,y)>maxI/2) AND (in2(x,y)<maxI/2))?maxI:0, where  maxI - maximal value of intensity
Video_ORout(x,y)=max(in1(x,y),in2(x,y))
Video_ORBout(x,y)= ((in1(x,y)>maxI/2) OR (in2(x,y)<maxI/2))?maxI:0, where  maxI - maximal value of intensity
Video_SUBout(x,y)=(in1(x,y) - in2(x,y))/2+maxI/2, where  maxI - maximal value of intensity
Video_EQ;out(x,y)=maxI if in1(x,y) = in2(x,y) and 0 otherwise, where  maxI - maximal value of intensity

This gadgets has no gadget setup dialog.
