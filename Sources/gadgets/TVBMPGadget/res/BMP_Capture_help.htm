  1. Data Processing – BMPCapture streams the bmp file loaded to it. Files can be loaded by typing file location. If file location ends with *.bmp, all the bmp files in that location will be streamed. BMPCapture streams one frame per data packet received in input pin. Frame rate can be determined in setup dialog (possible values: 1 - 25) and is only relevant if no data packets are received in input pin. If loop drop box is set to true, the bmp file\s will be streamed recursively.
