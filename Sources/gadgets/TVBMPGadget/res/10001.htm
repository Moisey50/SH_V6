BMPRender gadget
 
   Purpose:                  Saving to image files.

Input Formats:       CVideoFrame

Outputs Formats:  NONE

Duplex Formats:   NONE

Location in Gadget tree:  Renders\Files\Render\ (Grouped by type) or Files\Render\ (Grouped by description)

BMPRender gadget saves the received video data frames to bmp files. The video data frames received in input pin are compressed according to the value of 'FormatBMP' and saved in 'FilePath'.

Gadget setup dialog:

FilePath - sets the path where the bmp files will created. The name of the file coded corresponding C notation for printf. Allowed only one %d command, which encode the file number.

FormatBMP - determines the bmp format. Possible values: RGB32 \ yuv9.
