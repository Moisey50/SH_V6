WDMCapture gadget
 
   Input Formats:      transparent, used for software triggering

Outputs Formats:  CVideoFrame

Duplex Formats:    NONE

Location in gadget tree: Captures\Video\capture for grouped by type or Video\Capture by default

WDMCapture gadget controls analog or digital capture devices that use Windows Driver Model (WDM) drivers..

 

Gadget Setup has following option:

The view of setup depends on device you choose, but usually has two buttons:

  • Video Capture Filter
  • Video Capture Pin

The call corresponding dialogs from source driver, which quite different for different drivers, but usually first control parameters of image, and second - the format of input frame and frame rate. You also can choose one of three outputs format in the right radio button control, any input formats will be converted to one of them.
