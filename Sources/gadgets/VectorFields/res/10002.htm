ShowVectorField gadget
Input Formats: CContainerFrame(CVideoFrame,CUserDataFrame(VectorFields))
Outputs Formats: CContainerFrame(CVideoFrame,CUserDataFrame(VectorFields),CMetafileFrame)
Duplex Formats: NONE
Location in gadget tree: Filters\VectorFields\ for grouped by type or VectorFields\ by default

ShowVectorField converts input data  to MetafileFrame, which allows to display vector fields on the Video Renderer.

The work of the gadget with Vector Fields illustrated by this sample.

The gadget has no setup dialog.
