Outputs Formats:CContainerFrame(CVideoFrame,CUserDataFrame(VectorFields))
Duplex Formats: NONE
Location in gadget tree: Filters\VectorFields\ for grouped by type or VectorFields\ by default

MotionEstimate gadget determine motion vectors field, that describe the transformation of one image relative to reference image using block-matching algorithm. It can use in two modes with using prediction or not.

 

Gadget Setup has following option:

ROI - determine number pixels of searching, 1 - means that searching window will be on 2 pixels wide and higher then kernel window.

UsePrediction - use or not prediction vector fields

Base - the size of kernel window.
