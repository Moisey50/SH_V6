PlotGraph gadget
Input Formats:CFigureFrame or set of CArrayFrame
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Data\renderers for grouped by type or Data\renderers\ by default

PlotGraph gadget plots an input data as a 2D graph. It outputs the coordinates of mouse cursor in window in physical coordinates.

The sample of usage.

Gadget setup dialog:

Min - sets minimal value for Y coordinate

Max - sets maximal value for Y coordinate

Fit - sets autoscale for Y coordinate
