TVHistogram gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\color&brightness for grouped by type or Video\color&brightness\ by default

TVHistogram gadget outputs original video frame and histogram of luminance in the predefined rectangle as CFigureFrame. The rectangle is determined through control pin by following message:

Rect=x0,y0,x1,y1;

The sample of usage.

Gadget has no setup dialog.
