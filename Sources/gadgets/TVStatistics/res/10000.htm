LineProfile gadget
Input Formats:CVideoFrame
Outputs Formats:CContainerFrame(CVideoFrame,CFigureFrame)
Duplex Formats: in: CTextFrame
Location in gadget tree: Filters\Video\statistics\ for grouped by type or Video\statistics\ by default

LineProfile Gadget outputs original video frame and profile if luminance through the predefined line as CFigureFrame. The line is determined through control pin by following message:

Line=x0,y0,x1,y1;

The sample of usage.

Gadget has no setup dialog.
