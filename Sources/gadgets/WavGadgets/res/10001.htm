WaveCapture gadget
 
   Input Formats:      none

Outputs Formats:  CWaveFrame

Duplex Formats:    NONE

Location in gadget tree: Captures\Wave\ for grouped by type or Wave\ by default

Capture sound or other Waves from selected capture sound device and streams them to the output pin.

 

Gadget Setup has following option:

Select input device for sound capture from the list.
