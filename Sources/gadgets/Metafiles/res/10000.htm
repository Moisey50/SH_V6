WMF gadget
Input Formats:CVideoFrame, CRectFrame, CFigureFrame, CTextFrame, CQuantityFrame
Outputs Formats:CVideoFrame, CMetafileFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Metafiles\ for grouped by type or Metafiles\ by default

WMF gadget converts the data attached to the video stream to Windows Metafile (WMF) layer to enable its display in video renderer. The data will be drawn only in case of proper Attributes values set to data frame - it must set color and/or position and, probably, font size of drawing. For example, attributes for text drawing set:

color=0xFF0000;x=10;y=10;Sz=50;

Check sample for details.

Gadget has no setup dialog.
