ScrollCtrl gadget
Input Formats:any type, only if SynchroPin enabled
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Controls\Controls\ for grouped by type or Controls\ by default

ScrollCtrl gadget implements slider control. It generate text frame every time slider is moved or on any message come to Input pin.

 

Gadget setup dialog:

Min - minimal value for slider posiotion

Max - maximal value for slider position

Vertical - vertical slider if true, horizontal otherwise

MessageFormat - format for message in "C" printf format, only one %d allowed

SynchroPin - enables input pin
