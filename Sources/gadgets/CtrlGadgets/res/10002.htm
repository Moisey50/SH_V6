TextCaptureCtrl gadget
Input Formats:NONE
Outputs Formats:CTextFrame or CQuantityFrame
Duplex Formats: NONE
Location in gadget tree: Controls\Text\TextCaptureCtrl for grouped by type or Text\TextCaptureCtrl by default

TextCaptureCtrl gadget creates data packets according to the description set in setup dialog. Data packets are generated according the settings in setup dialog and sent to the output pit both automatically in the rate set in ‘FrameRate’ and manually when the ‘Apply’ button is pressed.

 

Gadget setup dialog has following parameters:

Quantity - determines if the data packet is of type quantity (True) or text (False). Possible values: true \ false.

NoSync - determines whether the data packet’s ID will be replaced with the ID of the data packet received in input pin. Possible values: true \ false.

Manual - determines whether the data packets will be sent automatically or manually (by pressing the button). Possible values: true \ false.

FrameRate - determines the frame rate of the automatically sent data packets. Possible values: 1 - 25.
