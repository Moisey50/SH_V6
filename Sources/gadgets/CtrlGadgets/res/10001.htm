CheckBoxCtrl gadget
Input Formats:NONE
Outputs Formats:CBooleanFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Video\ for grouped by type or Video\ by default

CheckBoxCtrl gadget sends true \ false logical data packets to its output pin when check box is checked \ unchecked. Once the checkbox is checked, ‘true’ logical data packet is sent to output pin. Once the checkbox is unchecked, ‘false’ logical data packet is sent to output pin. The checkbox is available only if the graph’s execution status is ‘start’.

 

Gadget setup dialog has following parameters:

 ButtonName - sets the text next to the checkbox.
