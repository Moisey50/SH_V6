Knob gadget
Input Formats:NONE
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Controls\Controls\Knob\ for grouped by type or Controls\Knob\ by default

Knob gadget sends text data packet to its output pin when pressed. Once the knob is pressed a text data packet with the text ‘ButtonName’ is sent to output pin. The knob is available only if the graph’s execution status is ‘start’.

 

Gadget setup dialog has following parameters:

 ButtonName - sets the text on the knob and the text sent to the output pin every time the knob is pressed.
