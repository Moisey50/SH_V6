Code128Reader gadget
Input Formats:CVideoFrame
Outputs Formats:CTextFrame

Code128Reader gadget translates Code 128 barcodes to alphanumeric symbols. Code128Reader searches for Code 128 barcode in the video data frame received in input pin, translates it to alphanumeric symbols and sends them to output pin in text data packet. The sample of code 128 barcode:
