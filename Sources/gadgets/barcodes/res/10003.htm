DataMatrixReader gadget
Input Formats:CVideoFrame
Outputs Formats:CTextFrame

Location in gadget tree: Filters\Video\recognition\ for grouped by type or Video\recognition\ by default

DataMatrixReader gadget translates data matrix barcodes to alphanumeric symbols. DataMatrixReader searches for data matrix barcode in the video data frame received in input pin, translates it to alphanumeric symbols and sends them to output pin in text data packet. Sample of Data matrix Barcode:
