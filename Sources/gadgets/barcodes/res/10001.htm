Code39Reader gadget
Input Formats:CVideoFrame
Outputs Formats:CTextFrame

Location in gadget tree: Filters\Video\recognition\ for grouped by type or Video\recognition\ by default

Code39Reader gadget translates Code 39 barcodes to alphanumeric symbols. Code39Reader searches for Code 39 barcode in the video data frame received in input pin, translates it to alphanumeric symbols and sends them to output pin in text data packet. Sample of the Code 39 barcode:
