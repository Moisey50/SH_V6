UPCReader gadget
Input Formats:CVideoFrame
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Video\recognition\ for grouped by type or Video\recognition\ by default

UPCReader gadget translates UPC barcodes to alphanumeric symbols. UPCReader searches for UPC barcode in the video data frame received in input pin, translates it to alphanumeric symbols and sends them to output pin in text data packet. Sample of UPC barcode:
