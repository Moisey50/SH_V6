NeedleDetector gadget
Input Formats:CVideoFrame
Outputs Formats:
  1. CVideoFrame
  2. CTextFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Video\recognition\ for grouped by type or Video\recognition\ by default

NeedleDetector gadget used for reading the value of pointer-type instrument. Usually used jointly with Locator gadget. After proper calibration gadget outputs video frame in which is shown pointer position and text frame with read value. It's suggested, that pointer is lighter then background.

 

Gadget has setup dialog:

Y Position - determine position of the line in which position of the pointer will be evaluated

First point

X Position - first calibration point in picture coordinates

Value - value, which corresponds that point

Second point

X Position - second calibration point in picture coordinates

Value - value, which corresponds that point
