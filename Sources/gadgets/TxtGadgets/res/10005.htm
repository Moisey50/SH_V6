NMEAParser gadget
Input Formats:CTextFrame
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Captures\Text\ for grouped by type or Text\ by default

NMEAParser parses NMEA 0183 messages. The input get output string from GPS device, usually through RS-232 interface, check it validity and return current coordinates in case of success.

For example, the string

$GPRMC,125504.049,A,5542.2389,N,03741.6063,E,0.06,25.82,200906,,,*17

will be converted to:

5542.2389N,03741.6063E

Gadget has no setup dialog.
