MatchTemplate gadget
Input Formats:CTextFrame
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Captures\Text\ for grouped by type or Text\ by default

MatchTemplate gadget allows extract data from text source an a base of text template. The syntax of template very simple:

* - skip whole text till find next text block

$varname$ - assign variable varname whole text, till find next text block in template.

For example if we have template:

*<A href=/cgi-bin/quake_stat.pl\?sta=*<td nowrap>$date$</td>

then from input text:

</tr><tr align=right bgcolor=CBD1CF background=''><td nowrap><A href=/cgi-bin/quake_stat.pl?sta=20134315>1</A></td><td nowrap>2013-12-14 04:06:14.0</td><td nowrap>35.68</td><td nowrap>140.72

gadget make output

date=2013-12-14 04:06:14.0

the template will be applied till rich end of file, so if there are several matches of template in the text, then gadgets makes several output text frames.
