HtmlRequest gadget
Input Formats:CTextFrame (just in case of CmdPin=TRUE)
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Captures\Text\ for grouped by type or Text\ by default

HtmlRequest gadget make http request to specified URL and return result as TextFrame to output frame. The URL can be specified in the setup dialog or can be received from other gadget through input pin.

 

Gadget setup dialog:

URL - write URL for http request here

request - write http request here

CmdPin - set true if you want to pass http request from another gadget, and false otherwise.

FrameRate - Rate of requests, available values from 0.1 to 10.0. Item visible only if CmdPin is false.
