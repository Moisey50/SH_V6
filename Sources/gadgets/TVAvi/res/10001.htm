AviCapture gadget
Input Formats:any type, used for synchronization, available only if Software trigger is allowed
Outputs Formats:CVideoFrame, CTextFrame, depends on format of source file
Duplex Formats: in: CTextFrame, out: CtextFrame
Location in gadget tree: Captures\Files\Capture\ for grouped by type or Files\Capture\ by default

AviCapture streames previously recorded Avi file. The number of output streams depends on number of the streams recorded in avi file, but can be reserved in setup dialog. Most of setup parameters can be controlled through control pin by command set/get parameter. Available following parameters:

  • FileName
  • SWTrigger
  • LoopFilm
  • ?

for example: set loopfilm(false)

Gadget setup dialog:

File name - select filename

Frame rate - frame rate for video output

Software trigger - allows using software triggering

Reserve output pins - reserve output pins in spite of the number streams recorded in avi file

Loop - loops film if selected.
