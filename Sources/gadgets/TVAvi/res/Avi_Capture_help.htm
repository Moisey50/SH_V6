  1. Setup Dialog:

AviCapture Gadget Setup Dialog

  1. Data Processing­ - AviCapture streams the avi file loaded to it. Files can be loaded via browse button or by typing file location.  If software trigger is checked, AviCapture streams one frame per data packet received in input pin. Frame rate can be determined in setup dialog (possible values: 1 to 25) only if software trigger is not checked. If loop check-box is checked, the avi file will be streamed recursively. A complete description of current setup dialog parameters is printed to a text data packet and sent to control pin as a response to text data packet containing the command ‘list’ received in control pin.

The value of a specific setup dialog parameter is printed to a text data packet and sent to control pin as a response to text data packet containing the command ‘get’ and the parameter’s name received in control pin:

-     get FileName - invokes the sending of ‘FileName’ parameter value to control pin.

-     get FrameRate - invokes the sending of ‘FrameRate’ parameter value to control pin. Possible values: 1 - 25.

-     get SWTrigger - invokes the sending of ‘SWTrigger’ parameter value to control pin. Possible values: true / false.

-     get LoopFilm - invokes the sending of ‘LoopFilm’ parameter value to control pin. Possible values: true / false.

The value of a specific setup dialog parameter can be altered as a response to text data packet containing the command ‘set’, the parameter’s name and its value received in control pin, for example: set LoopFilm(true) will set the LoopFilm characteristic to true. The list of parameters and their possible values is given above.
