DShowCapture gadget
Input Formats:NONE
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Capture\Files\Capture\ for grouped by type or Files\Capture\ by default

DShowCapture gadget reads lots types of media files. The full list of files types depends on installed in the machine DirectShow filters. Most of setup parameters can be controlled through control pin by command set/get parameter. Available commands:

FileName
SWTrigger
LoopFilm
Length
Position
?

For example:
set loopfilm(false)

Gadget setup dialog has following parameters:

FileName - path to source file

LoopClip - set true to loop file playing.

SoftwareTrigger - use trigger for frames output.
