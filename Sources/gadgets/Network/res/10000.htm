NWSocket gadget
Input Formats:any type of data
Outputs Formats:any type of data
Duplex Formats: NONE
Location in gadget tree: Filters\Communication\ for grouped by type or Communication\ by default

NWSocket gadget enables data duplex exchange between different applications/computers by TCP/IP protocol. Pair of gadget can establish connection if one set in server, but another in client mode.

 

Gadget has two types of setup dialog:

For server mode:

Mode - set Server for server mode

Port - sets TCP/IP port

StopWhenGraphStoped - if true - gadget stops receiving data when graph stops, gadget continue receiving data otherwise

and for client mode:

Mode - set Server for server mode

ServerAddress - sets IP address of the server

Port - sets TCP/IP port

DataType - determines the type of the data packets that will be exchanged between the server and the client

StopWhenGraphStoped - if true - gadget stops receiving data when graph stops, gadget continue receiving data otherwise
