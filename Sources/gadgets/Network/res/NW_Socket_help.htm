  2. Location in Gadget tree:

    2. Grouped by description: Communication \ NWSocket

  1. Setup Dialog -  

NWSocket Server Setup Dialog

NWSocket Client Setup Dialog

-    Server Setup Dialog

·     Mode - determines whether the gadget will be used as server or client.

·     Port - port number which is used, along with the server’s host computer IP address as the server address.

-    Client Setup Dialog

·     Mode - determines whether the gadget will be used as server or client.

·     Server address - IP address of the server’s host computer.

·     Port - port number.

·     Data type - determines the type of the data packets that will be exchanged between the server and the client.

-    Text - text data packet

-    Data packets - any other data packet type

 

  1. Data Processing – If 2 NWScoket gadgets are conjoined (i.e. port number is identical and the server’s host computer IP address is set in the client setup dialog), data packets received in client’s input pin will be sent to  the server’s output pin and data packets received in server’s input pin will be sent to the client’s output pin.
