Lines gadget
Input Formats:CVideoFrame, CFigureFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Video\recognition\ for grouped by type or Video\recognition\ by default

Lines seeks straight line segments between different textures in the video data frame. The processed video data frame with straight segments coded in CFigureFrame is streamed to the output pin.

Sample of usage.

Gadget has no setup dialog.
