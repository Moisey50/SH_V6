FLWCapture gadget
Input Formats:any type, used for synchronization, available only if Software trigger is allowed
Outputs Formats:any type, depends on format of source file
Duplex Formats: in: CTextFrame, out: CtextFrame
Location in gadget tree: Captures\Files\Capture\ for grouped by type or Files\Capture\ by default

FLWCapture streames previously recorded FLW file. Most of setup parameters can be controlled through control pin by command set/get parameter. Available commands:

  • getmax - return max frame nmb
  • seek <frnmb>
  • getmaxtime - return time for last frame
  • seektime <time>
  • get/set FileName
  • get/set LoopFilm
  • get/set SWTrigger
  • get/set ConstRate
  • get/set FrameRate
  • ?

for example: set loopfilm(false)

Gadget setup dialog:

File name - select filename

Frame rate - frame rate for video output, if not checked - the timing recorded in FLW file will be played.

Software trigger - allows using software triggering

Loop - loops film if selected.

AutoRewind - sets auto rewind to start when played to end.
