SlideView gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters renderers\Video\ for grouped by type or Video\renderers\ by default

SlideView gadget is multi-frame video renderer gadget. The length of the shown frames can be set through setup. The gadget can work in shift or circle replace mode. Gadget make as output the selected frame in the stack stored frames. The output makes at event of selection new frame, or in case of replacing old frame by new in selected position.

Gadget setup dialog:

Scale - The scale of displayed image

Monochrome - Show greyscale if true

ShiftMode - Shift mode if true, circle replace - otherwise

Rescale - Rescale every image to width TargetWidth if true

TargetWidth - valid only if Rescale is true, every input frame in slide view can be rescaled to this width

FramesLen - length of the set of frames in a view

FramesInRow - set the length of the images in one row, number of the rows can be calculated by FramesLen/FramesInRow

ShowMetafiles - show or not metafiles in slide represantation
