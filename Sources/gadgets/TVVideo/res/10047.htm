VideoHMeander gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\convolution\ for grouped by type or Video\convolution\ by default

VideoHMeander makes convolution of every row of input image with square shaped pulse. It allows to seek darker shapes with target width.

 

Gadget setup dialog has one parameter - QSize - it determine the width of the pulse.
