VideoEdgeDetector gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\color&brightness for grouped by type or Video\color&brightness\ by default

VideoEdgeDetector is aggressive edge detector. It replace each point luminance with maximal absolute difference between it's original luminance and the luminance of it's neighbours.

 

Gadget has no setup dialog.
