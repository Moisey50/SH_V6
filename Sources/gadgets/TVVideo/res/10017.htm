PseudoColor gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\color&brightness for grouped by type or Video\color&brightness\ by default

PseudoColor Gadget converts initially grey scale image to color image by mapping the each pixel luminance value to a color according to table. The conversion is illustrated by the picture below:

Fig. The conversion of grey levels to colored levels.

 

Gadget setup dialog:

Allows to set input range of conversion by setting Max and Min values of luminance.
