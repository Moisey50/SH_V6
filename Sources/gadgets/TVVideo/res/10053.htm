ColorMap gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\math\ for grouped by type or Video\math\ by default

ColorMap differentiates between different colors. The U-V color plane is divided to 9 sectors according to the vertical and horizontal shifts of the horizontal and vertical dividing lines. Pixel intensity is assigned to each sector according to the following tables:

SectorPixel Intensity
00
150
275
3100
4125
5150

For each pixel of the received video data frame, its color location in the U-V color plane is determined and its new pixel intensity is set accordingly. The converted video data frame is streamed to the output pin.

Gadget setup dialog has two parameters:

  • U_Thres - sets the horizontal shift of the vertical lines dividing the U-V color plane to both sides. Possible values: 0 - 255.
  • V_Thres - sets the vertical shift of the horizontal lines dividing the U-V color plane to both sides. Possible values: 0 - 255.
