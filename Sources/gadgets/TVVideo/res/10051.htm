ScanLine gadget
Input Formats:CVideoFrame
Outputs Formats:CContainerFrame(CFigureFrame*3,CTextFrame)
Duplex Formats: in: CTextFrame
Location in gadget tree: Filters Diagnostic\ for grouped by type or Filters\Diagnostic\ by default

ScanLine displays the profile of the image at given line (Y coordinate). The line (Y coordinate) is defined by a text data packet received in control pin of the following format: selected=true;x=<int>;y=<int>;

Video stream must be connected to input pin.

The output data (figure packet) should be integrated with the original video stream using Aggregator gadget with the following setup: pin1=;pin2=;

The Aggregator gadget must be connected to WMF to display the figure data.

Load the sample. Click to output window to set row profile to display.

 

Gadget has no setup dialog.
