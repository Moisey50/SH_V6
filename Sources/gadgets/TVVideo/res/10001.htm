VideoRender gadget
 
   Input Formats:      CVideoFrame

Outputs Formats:  CTextFrame

Duplex Formats:    NONE

Renders video frame. Has two pins inpiut and output.

  • Input pins receive input video frame. In video frame can be changed attributes to x=NNN;y=MMM; - they allows to control center of display image
  • Output pins informs about mouse events in Render's focus.
    In case of focus moving pin send message "x=<xPos>; y=<yPos>; I=<Intensuty value under the mouse>; U=<U value under the mouse>; V=<V value under the mouse>;"
    In case of click mouse message will be "Selected=true; x=<xPos>; y=<yPos>; I=<Intensuty value under the mouse>; U=<U value under the mouse>; V=<V value under the mouse>;"
     

 

Gadget Setup has following option:

Scale - allows change scale of displaying - x1,x2,x4, Fit Window

Monochrome - option allows change color-monochrome type of displaying.

LineSelection - allows or disables line selection with left mouse button.

RectSelection - allows or disables rectangle selection with left mouse button.
