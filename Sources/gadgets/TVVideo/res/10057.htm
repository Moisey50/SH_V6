SquareFilter gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\frequency\ for grouped by type or Video\frequency\ by default

SquareFilter gadget is a kind of blur image filter, each point luminance is replaced with average one of predefined square area.

 

Gadget setup dialog has following parameters:

Box - can be one of box3x3, box5x5 and box7x7, they define the area of averaging.
