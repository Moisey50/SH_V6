VideoLowPass1DV gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\frequency\ for grouped by type or Video\frequency\ by default

VideoLowPass1DV is one of the one dimensional filters. It use same algorithm as 2D filter VideoLowPass gadget, but make low-pass filtering just in vertical direction. That allows to reduce noise, but saves vertical elements in the image. It has the same setup dialog as VideoLowPass gadget.
