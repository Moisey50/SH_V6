VideoSimpleBinarize gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\binarizing\ for grouped by type or Video\binarizing\ by default

VideoSimpleBinarize Gagdet converts an input image of some grey or color levels to black and white. This thresholding operation makes all pixels of the image which luminance lower threshold level - black, and  white otherwise.

Gadget setup dialog:

Level - sets threshold level.
