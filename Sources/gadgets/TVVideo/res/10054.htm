Any2Yuv9 gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters filters\Video\conversion\ for grouped by type or video\conversion\ by default

Any2Yuv9 gadget converts any format Video Frame to yuv9 format, which is most supported in Studio.

 

Gadget has no setup dialog.
