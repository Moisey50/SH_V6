VideoNormalizeEx gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\color&brightness for grouped by type or Video\color&brightness\ by default

VideoNormalizeEx Gadget makes the intensity histogram stretching of the input image from predefined range to full dynamic range. The range for source  image defined through setup dialog.

 

Gadget setup dialog:

Min - defines the start of source range.

Max - defines the end of source range.
