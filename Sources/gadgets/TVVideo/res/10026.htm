VideoHighPass1DH gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\frequency\ for grouped by type or Video\frequency\ by default

VideoHighPass1DH is one of the one dimensional filters. It use same algorithm as 2D filter VideoHighPass gadget, but make high-pass filtering just in horizontal direction. That allows to emphasize vertical elements in the image. It has the same setup dialog as VideoHighPass gadget.
