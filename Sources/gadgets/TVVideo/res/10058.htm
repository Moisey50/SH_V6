Rotate gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: in: CTextFrame, out:CTextFrame
Location in gadget tree: Filters \Video\frame\ for grouped by type or Video\frame\ by default

Rotate gadget rotate entire image pr it's rectangle part on arbitrary angle. The mode, angle and rectangle can be set or through setup dialog, or through control pin.

 

Gadget setup dialog has following parameters:

Angle - angle of rotation in degrees.

RotateRgn - can be one of the following: RotateFrame(0) , RotateRegion(1)

Rect - this parameter determine rectangle region for rotation, this parameter valid only if RotateRgn set to RotateRegion value.
