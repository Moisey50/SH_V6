VideoAverage gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame, CQuantityFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\average\ for grouped by type or Video\average\ by default

VideoAverage gadget average the sequence of video frame in a time, according to the selected averaging method. The processed video frame is streamed to output pin. Also streamed out the average luminance over the frame.

 

Gadget setup dialog has following options:

Mode - set the averaging method. Possible values:

  • InfiniteUniform - average over all the frames till current frame, each frame has the same weight.
  • InfinitePastFading - averaging over all the frames till current frame, average till previous frame weights 75% and current frame weights 25%.
  • SlidingWindow - average over the last ‘FramesRange’ frames, each frame has the same weight.

FramesRange - sets the number of frames SlidingWindow averages over. Available only if mode is SlidingWindow. Possible values: 1 - 1000.
