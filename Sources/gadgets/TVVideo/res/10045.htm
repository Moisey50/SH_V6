Block8 gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\frame\ for grouped by type or Video\frame\ by default

Block8 changes the size of the video data frame received in input pin so that it will be dividable by 8. Both the horizontal and vertical sizes of the video data frame received in input pin are diminished until they are dividable by 8. The diminished (or original if it was dividable by 8) video data frame is streamed to output pin.

 

Gadget has no setup dialog.
