VideoPercentBinarize gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\binarizing\ for grouped by type or Video\binarizing\ by default

VideoPercentBinarize Gadget converts input image to output two color (B/W) image on a base of predefined percents of white pixels.

 

Gadget setup dialog:

Percent - set percentage of white pixels on a result image.
