VideoCutRect gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: CTextFrame or CRectFrame
Location in gadget tree: Filters \Video\cut\ for grouped by type or Video\cut\ by default

VideoCutRect gadget cuts a fragment image from the entire input image. The parameters of cut rectangle can be passed to gadget or with Setup Dialog, or through control pin. Control pin can accept the target rectangle as binary, in CRectFrame data frame, or as text, in  CTextFrame data frame. In case of text format there are two commands:

ROI=x0,y0,x1,y1; or Rect=x0,y0,x1,y1; - sets rectangle directly

Center=x,y; - move current rectangle to new position.

 

Gadget setup dialog:

allows set up the position of the target rectangle or with an edit box controls, or by mouse in a preview window. The CheckBox 4 pixels step sets 4 pixels step, if checked, or 1 pixel step - otherwise. It affect only on grey scale images, color images always has 4 pixel step of the target rectangle edges.
