MergeLayers gadget
Input Formats:CContainerFrame(CVideoFrame,CMetafileFrame...)
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters Other\Video\Conversion\ for grouped by type or Video\Conversion\ by default

MergeLayers gadget merges all the video related data layers inside a container frame to a single video data frame, so all metafile drawings will be drown over the single video frame, and can be written with avi or bmp renders.

 

Gadget has no setup dialog.
