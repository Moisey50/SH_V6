Render3DPlot gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: in CTextFrame, out CTextFrame
Location in gadget tree: Filters \Video\renderers\ for grouped by type or Video\renderers\ by default

Render3DPlot gadget represent input video frame as 3D plot, where the luminance is set in Z-direction. The sample of source image and result shown in the picture:

 

Gadget setup dialog:

Parameters:

  • Scale - allows rescale image in the window or fit image to window frame
  • Monochrome - make grey scale image if true
  • LineSelection - allows select a line in the image if true
  • RectSelection - allows select a rectangle in the image if true
  • Alpha - rotate plot
  • Beta - rotate plot
  • Grid - change distance between grids
  • RotZ - rotate plot
