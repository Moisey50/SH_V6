CustomFilter gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\color&brightness for grouped by type or Video\color&brightness\ by default

CustomFilter gadget. By specifying a small table of numbers, many simple effects can be achieved. The result luminance of the pixels will be calculated as sum of neighbours with corresponding weight coefficients.

 

Gadget setup dialog:

Box - Size of filter matrix, can be 3x3,5x5 and 7x7

Matrix0...N - rows of matrix of the pixel weights.

Offset - value, which will be added to result.
