VideoSharpen gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\frequency\ for grouped by type or Video\frequency\ by default

VideoSharpen gadget allows to make the sharpness of image elements. However it can increase image noise.

 

Gadget setup dialog has one parameter - ConvParameter - it can be from 0 to 1000, and determine the radius of the effect in arbitrary units.
