VideoHighPass gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\frequency\ for grouped by type or Video\frequency\ by default

VideoHighPass perform fast emboss operation (applying High-pass filter) on input video frame. 

 

Gadget setup dialog:

ConvParameter - can be from 0 to 1000, and determine the radius of the convolution in arbitrary units.
