MassBinarize gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\binarizing\ for grouped by type or Video\binarizing\ by default

MassBinarize gadget converts input grey or color image to the two level image, as threshold is used average luminance over whole image. The value of threshold can be corrected by corresponding ratio defined in setup dialog.

 

Gadget setup dialog:

  Ratio - set correction value here.
