Video81PassFilter gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\edges\ for grouped by type or Video\edges\ by default

Video81PassFilter is kind of edge detector video filter. Each illuminance of the point replaced with absolute difference of original illuminance and average illuminance of all it's neighbours. It correspond the following custom filter:

-1 -1 -1
-1  8 -1
-1 -1 -1

Gadget has no setup dialog.
