ColorBalance gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\color&brightness for grouped by type or Video\color&brightness\ by default

ColorBalance gadget balances the U and V color components of the video frame automatically. Determines the shifts required in the U and V color components and performs these shifts on the video data frame received in input pin. The balanced video data frame is streamed to output pin. Note that the format of the received video must be YUV9 (video streams of the format Y8 and Y16 are streamed to output pin without processing).

 

Figure: UV Color Space

 

Gadget has no setup dialog.
