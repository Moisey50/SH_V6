VideoResample gadget
Input Formats:CVideoFrame
Outputs Formats:CVideoFrame
Duplex Formats: NONE
Location in gadget tree: Filters \Video\color&brightness for grouped by type or Video\color&brightness\ by default

VideoResample gadget resample the input image to other size. Output sizes are fitted to closest dividable 4 value.

 

Gadget has setup dialog with two parameters - Width and Height, which determine target sizes for output image.
