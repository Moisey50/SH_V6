CompareFilter gadget

Input Formats:     CQuantityFrame

Outputs Formats: CBooleanFrame – result of comparing with predefined value

Duplex Formats:   NONE

This gadget compare input quantity with predefined value.

Example:

In the example input frames counts, the count of the frames compared with predefined value 10 while it become greater or equal then 10, and then open Switch gadget to pass frames.
