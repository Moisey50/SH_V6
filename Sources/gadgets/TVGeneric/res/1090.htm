RenderID gadget
Input Formats:CDataFrame
Outputs Formats:NONE
Duplex Formats: NONE
Location in gadget tree: Filters \Control\Diagnostic\ for grouped by type or Diagnostic\ by default

RenderID gadget displays the ID of the received data packet alongside the Graph Time, in which the data packet was received, the Type of the data packet and the total number of data packets received since last stop.

 

Gadget has no setup dialog.
