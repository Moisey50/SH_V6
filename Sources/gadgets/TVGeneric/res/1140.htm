Uniselector gadget
Input Formats:CQuantityFrame
Outputs Formats:CBooleanFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Generic\ for grouped by type or Generic\ by default

Uniselector gadget calculates the value of the quantity data packet received in control pin modulo the number of output pins and sends logical true data packet to the corresponding output pin and logical false data packet to all the other output pins.

 

Gadget setup dialog has following parameters:

OutputPins - sets the number of output pins and the modulo argument. Possible values: 1 - 50.
