Delay gadget
Input Formats:CDataFrame (any type allowed)
Outputs Formats:CDataFrame (same as received)
Duplex Formats: NONE
Location in gadget tree: Filters\Diagnostic\ for grouped by type or Diagnostic\ by default

Delay gadget delay data packet sending to output pin with defined delay time.

 

Gadget setup dialog has following parameters:

Delay - defines the period of time in which the data packet is delayed. Possible values: 1 - 99 milliseconds.
