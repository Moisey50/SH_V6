Switch gadget

Input Formats:     Any

Outputs Formats: Same data packet. Will be passed through filter in dependence on data on control pin.

Duplex Formats:   CBooleanFrame – if true – the switch pass the data from input to output. If ID of the frame != 0, then this data frame ID must be same and for input (sync mode), otherwise the state of the key will be saved – and it will be open or close till next control data packet.

This gadget control passing data.

Example:

In the example input frames counts, the count of the frames compared with predefined value 10 while it become greater or equal then 10, and then open Switch gadget to pass frames.

Gadget has no setup dialog.
