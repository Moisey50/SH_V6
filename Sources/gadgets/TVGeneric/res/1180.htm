LabelFilter gadget
Input Formats:any type of the data
Outputs Formats:can be any
Duplex Formats: NONE
Location in gadget tree: Filters\Generic\ for grouped by type or Generic\ by default

LabelFilter gadget filters the received data packets by set of rules: according DataFrame's label or/and dataframe type.

 

Gadget  setup dialog has following options:

Label - sets the value of Label parameter according to which the data packets will be filtered.

Type - sets type to select

Available - this listbox filled with all available labels in data stream. Selection one of them lead to appending it to Label string.
