ChangeID gadget
Input Formats:any type allowed
Outputs Formats:same as received
Duplex Formats: NONE
Location in gadget tree: Filters\Generic\ for grouped by type or Generic\ by default

ChangeID gadget changes data packet ID by adding or subtracting a constant from original ID.

 

Gadget setup dialog has following parameters:

Add_to_ID - sets the constant that will be added (can be set to negative numbers) to data packet original ID.
