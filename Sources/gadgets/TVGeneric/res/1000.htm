GenericCapture gadget
 
   Input Formats:     ANY

Outputs Formats: See description

Duplex Formats:   NONE

GenericCapture gadget allows generate different data frames. They are:

  • nulltype
  • logical
  • rectangle
  • text
  • quantity
  • vframe
  • figure

Also, it is possible to set label and attributes to frame, disable or enable sync flag and enable registered flag.

Gadget can work in several modes. They are:

  • Manual or automatic. Automatic can be with external software synchronization and without it. In manual mode frame will sent just in case pressing of button "Send" in the body of gadget.
  • Transparent or replace mode. In transparent mode frame passed through gadget and only flags, attributes or labels will be changed. In replace mode frame will be changed with corresponding settings of gadget.

Some types of frames in replace mode required specific setup texts in Message field of gadget setup.

They are:

  • For rectangle frame that must be rect=<left rectangle corner>, <top rectangle corner>, <right rectangle corner>, <bottom rectangle corner>;
    for example: rect=10,10,100,100;
  • For logical frame that must be Bool=<true>,<false>;
    for example Bool=true;
  • For text frame that must be TextData=<text string>;
    for example: TextData=bla bla bla; there are some replacements codes for some chars:
    \ - \\
    ; - \s
    = - \e
    so, to send "bla;bla;bla" you must enter TextData=bla\sbla\sbla;
  • For quantity frame that must be Quantity=<value>;
    for example: Quantity=13.33; or Quantity=13;
  • For video frame that must be VidFrame=<path>\<file mask>;
    for example: VidFrame=D:\1\000?.bmp; or VidFrame=D:\1\*.bmp;
  • For figure frame that must be Points=x0,y0,x1,y1,...,xN,yN;
    for example: Points=1,1,20,20,40,1;

You also can change or set attribute for a frame, for example to set color and position of text for text frame write color=0xFF0000;x=100;y=100; in attribute field of setup dialog.
