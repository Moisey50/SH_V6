Sink gadget
Input Formats:Any type of the data
Outputs Formats:CContainerFrame
Duplex Formats: NONE
Location in gadget tree: Other\Generic\ for grouped by type or Generic\ by default

Sink gadget sends the received data packet and the pin number from which it was received to output pin. The received data packet is inserted to a container frame along with a quantity frame whose value equals the pin number from which it was received. The container frame is sent to output pin.

 

Gadget setup dialog has following parameters:

InputPins - determines the number of input pins.
