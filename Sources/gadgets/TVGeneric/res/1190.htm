GenericRender gadget
Input Formats:any data type
Outputs Formats:NONE
Duplex Formats: NONE
Location in gadget tree: Controls\Generic\ for grouped by type or Generic\ by default

GenericRender gadget use for rendering data procedure encapsulated in CDataFrame. The prototype of procedure is following:

bool Draw(CDC * dc, RECT rc);

overwrite it for any of custom data types, for rendering in this gadget.

 

Gadget has no setup dialog.
