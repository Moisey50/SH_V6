LPTBitRender gadget
Input Formats:CDataFrame
Outputs Formats:NONE
Duplex Formats: NONE
Location in gadget tree: Controls\Generic\ for grouped by type or Generic\ by default

LPTBitRender sends one 1 ms pulse on pin 1 of LPT port on each data packet of any type, which will be received on input pin.

Attention: The gadget is supported just under WinXP.

Gadget setup dialog has following parameters:

Port - can be choosen the LPT port to send data.
