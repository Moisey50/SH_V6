Null gadget
Input Formats:any type of the frame
Outputs Formats:same as input
Duplex Formats: NONE
Location in gadget tree: Other\Generic\ for grouped by type or Generic\ by default

Null gadget just transparently transmit any type data packet to output. Can be used as isolator so that if a certain gadget needs to be replaced, all its connectors won’t be deleted if they are connected through the Null gadget.

 

Gadget has no setup dialog.
