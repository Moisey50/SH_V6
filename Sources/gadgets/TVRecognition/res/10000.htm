OCR gadget
Input Formats:CVideoFrame
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Video\recognition\for grouped by type or Video\recognition\ by default

OCR gadget recognizes alphanumeric symbols and sends them in text data packet to output pin. The recognition is done according to font data base that can be elaborated to include additional fonts. Note, that Binarizetion must be applied on the video data stream before OCR processing and that the size of each alphanumeric symbol should approximately be 30x30 pixels. VideoEnlarge and VideoDiminish gadgets can be used to adjust the size.

 

Gadget setup dialog has one parameter:

LibFileName - path to external OCR library file, the internal one is used if path is not specified.
