Command gadget
Input Formats:CTextFrame or any, in this case used just for synchronization
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Text\ for grouped by type or Text\ by default

Command gadget allows to access to command-line interpreter of Windows OS. The gadget executes commands and send result to output as text frame. The full command send to interpreter is constructed as Prefix + Command + CTextFrame.

 

Gadget setup dialog:

Prefix - usually "cmd /C"

Command - any of cmd.exe command. Full list of the commands can be found here.
