MODIOCR gadget
Input Formats:CVideoFrame
Outputs Formats:CTextFrame
Duplex Formats: NONE
Location in gadget tree: Filters\Video\recognition for grouped by type or Video\recognition by default

MODIOCR gadget is OCR gadget, which uses for OCR Microsoft© MODI (Microsoft Office Document Imaging). To install MODI refer to this recommendations. Gatget gets video frame from input, and outputs recognized text as CTextFrame.

 

Gadget setup dialog:

OverwriteDPI - overwrite image resolution settings, recommend to overwrite

X_DPI - horizontal resolution

Y_DPI - vertical resolution.
